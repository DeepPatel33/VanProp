# VanProp
Users find it challenging to evaluate properties, comprehend market trends, and make wise real estate selections in Vancouver due to the distributed structure of property information. This program centralizes property tax data and offers an intuitive interface for tracking, comparing, and locating properties.


Dataset: property tax report

https://opendata.vancouver.ca/explore/dataset/property-tax-report/information/?disjunctive.neighbourhood_code

Name: Van Prop

Audience:

homebuyer who are researching property and neighbourhood 
Real estate investors
residents who want to know their property value
Current problem: 
Users find it challenging to evaluate properties, comprehend market trends, and make wise real estate selections in Vancouver due to the distributed structure of property information. This program centralizes property tax data and offers an intuitive interface for tracking, comparing, and locating properties.

Functionality:
Searching property, neighbourhood and property value.
Comparing property.
Filter properties by various critirea.
Add properties to watchlist, create saved searches, add custom notes to properties.
Search and display property details, view filtered property lists, show neighborhood statistics.
Modify watchlist notes, update saved search filters, edit comparison lists.
Remove properties from watchlist, delete saved searches.
